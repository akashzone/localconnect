# LocalConnect - Local Business Marketplace

LocalConnect is a MERN Stack web application that connects local business owners with student developers. It provides a platform where businesses can post website or software development requirements, and students can apply to work on those projects, helping businesses establish an online presence while giving students real-world experience.

## Tech Stack

- React
- Node.js
- Express.js
- MongoDB
- Tailwind CSS
